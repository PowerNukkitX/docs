# Configuration du Proxy

Utilisez un proxy pour transférer rapidement les joueurs entre plusieurs serveurs,
ce qui est particulièrement important pour configurer un groupe de serveurs. Dans l'édition Bedrock,
le proxy le plus populaire actuellement est [WaterdogPe](https://github.com/WaterdogPE/WaterdogPE).
PNX a créé une version adaptée pour celui-ci : [WaterdogPe-PNX](https://github.com/PowerNukkitX/WaterdogPE-PNX).
![waterdogpe](/img/proxy-config-waterdogpe.png)

Ensuite, nous allons expliquer comment utiliser WaterdogPe avec PNX.

## Utilisation

Vous devez simplement ajuster le fichier `nukkit.yml` de chaque sous-serveur PNX en définissant `waterdogpe=true` pour utiliser WaterdogPe.
