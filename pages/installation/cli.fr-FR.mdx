# Installation de PowerNukkitX avec le CLI (recommandé)

<Callout type={"warning"}>Le nouveau CLI est encore en beta et peut ne pas fonctionner correctement. Si vous rencontrez des problèmes, vous pouvez toujours signaler un problème sur le <a href={""}>GitHub Issue</a> et utiliser <a href={""}>l'installation manuelle</a>.</Callout>

## Prérequis
- Java 21 ou supérieur [Télécharger](https://adoptopenjdk.net/)
- Node.js 20 [Télécharger](https://nodejs.org/en/download/)

## Utilisation
Dans cette utilisation, nous utiliserons le nom `server` pour le dossier du serveur, mais vous pouvez utiliser n'importe quel nom.
<Steps>
    ### Créez un dossier
    <Tabs items={['linux/debian']}>
        <Tabs.Tab>
            ```bash
            mkdir server
            cd server
            ```
        </Tabs.Tab>
    </Tabs>
    <FileTree>
        <FileTree.Folder name="server" defaultOpen children="" />
    </FileTree>

    ### Exécutez le CLI
    <Callout type={"error"}>Attention, la commande doit être exécutée dans le dossier que vous venez de créer, sinon le CLI ne fonctionnera pas correctement.</Callout>
    ```bash
    npx pnx-cli
    ```
    Suivez les instructions pour installer votre serveur plus facilement.
</Steps>
