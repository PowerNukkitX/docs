## System Requirements
- Windows 10/11
- macOS 10.15 or later
- Linux (Ubuntu 20.04, Fedora 32, Debian 10/11)

## Java Requirements
Java 21 is essential to run the program. You can download it for Windows/MacOS: [Java 21](https://sdkman.io/jdks).

<details>
    <summary>For Linux</summary>

    To install <a>Java 21 on Linux</a>, you can follow these steps:

    <Steps>
        ### Open a terminal.

        ### Update packages

        <Tabs items={['with sudo', 'without sudo']}>
            <Tabs.Tab>
                ```bash
                sudo apt update
                ```
            </Tabs.Tab>
            <Tabs.Tab>
                ```bash
                apt update
                ```
            </Tabs.Tab>
        </Tabs>

        ### Install JDK 21

        <Tabs items={['with sudo', 'without sudo']}>
            <Tabs.Tab>
                ```bash
                sudo apt install openjdk-21-jdk
                ```
            </Tabs.Tab>
            <Tabs.Tab>
                ```bash
                apt install openjdk-21-jdk
                ```
            </Tabs.Tab>
        </Tabs>

        ### Verify the installation

        ```bash
        java -version
        ```

        If everything went well, you should see the Java version displayed.
    </Steps>

</details>
