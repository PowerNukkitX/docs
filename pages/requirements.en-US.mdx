# Requirements

## System Requirements
- Windows 10/11
- macOS 10.15 or later
- Linux (Ubuntu 20.04, Fedora 32, Debian 10/11)

## Java Requirements
Java 21 est indispensable pour exécuter le programme. Vous pouvez le télécharger [ici](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html).
